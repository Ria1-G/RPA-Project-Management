# RPA-Project-Management
RPA Project Management for Real Estate (WBS, Gantt, Cost Analysis)
# > Academic Project showcasing Project Management & Automation skills

## 📌 Overview
This project presents a complete Project Management plan for implementing Robotic Process Automation (RPA) in a real estate company.

It includes:
- Work Breakdown Structure (WBS)
- Gantt Chart (Project Schedule)
- Resource Allocation
- Network Diagram
- Cost Analysis
- Critical Path Analysis

---

## 🧠 What is RPA?
Robotic Process Automation (RPA) is software that automates repetitive tasks such as data entry, system updates, and workflow processing.

In real estate, it helps:
- Automate lease processing
- Reduce manual data entry
- Improve compliance and accuracy
- Save operational costs
